# Bem-vindo ao Storybook

Bem-vindo ao Storybook do seu projeto!
Aqui você ira visualizar os componentes existentes neste projeto, seu funcionamento e suas propriedades.

## Componentes disponíveis
- Explore a lista de componentes na barra lateral
- Veja a documentação de cada componente e os exemplos de uso
- Veja propriedades utilizadas por estes componentes.

## Começando
Para utilizar um componente em seu projeto, siga as instruções e exemplos nos stories individuais.

Neste projeto Front-end grande parte das propriedades , estados e funções dos componentes estão descritos em Context API na pasta de 'Context'.

### Atenção :
 Alguns componentes precisam de um backend da aplicação funcionando para que funcionem corretamente, apenas alguns componentes foram criados mocks de dados para que sejam visuais.

 Ative o backend em modo desenvolvedor com o banco de dados de desenvolvimento para visualizar a documentação do storybook sem causar erros.

 Alguns estados e propriedades são passados para os componentes atráves de Mocks do Context utilizado, sendo substituido os valores do Context por um definido nos 'agrs' do componente de stories do storybook.
